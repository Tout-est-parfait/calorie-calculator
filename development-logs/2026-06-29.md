- [x] **Phase 5：科学建议引擎完善**
  - 创建 `js/advisor.js`（建议引擎独立模块，从 app.js 提取并大幅增强）
  - 5 个分析维度：热量 / 碳水 / 蛋白质 / 脂肪 / 综合营养均衡
  - 热量分析：严重超标(>120%) / 超标(>105%) / 适中(85-105%) / 不足 — 4 级精细判断
  - 碳水分析：偏高(>65%) / 偏低(<45%) — 含复合碳水替代建议
  - 蛋白质分析：严重不足(<8%) / 偏少(<12%) / 偏高(>25%) — 含高蛋白食物推荐
  - 脂肪分析：偏高(>35%) / 偏低(<15%) — 含低脂替代建议
  - 运动消耗方案：7 种运动参考数据（跑步/快走/游泳/跳绳/骑行/HIIT/爬楼梯/瑜伽），热量超标时自动推荐 top 5
  - 食物补充建议：热量不足时推荐高营养密度食物
  - CSS 新增：exercise-list/row、food-tag-list/row/tag
